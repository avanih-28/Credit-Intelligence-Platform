# Credit Intelligence Platform 💳

A full-stack credit scoring web application built using modern web technologies.

## What it does
- Calculates transparent credit scores for users
- Provides financial insights and recommendations
- Supports users across multiple regions

## Tech Stack
- Frontend: React, Vite
- Backend: Express, TypeScript
- Database: Firebase

## What I learned
- How full-stack applications are structured
- How frontend and backend communicate via APIs
- How credit scoring logic can be implemented in code

🚧 Work in progress — built as part of a learning project
